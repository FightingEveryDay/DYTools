# DYTools
主要思想是简化oc过长的方法名.和增加一些经常用到的工具,但是oc语言中并没有的功能
QQ群：553633494
# 增加cocoapods管理
```
 pod 'DYTools'
```
### 即可使用最新第三方DYTools

# 导入头文件即可使用 
```apple js
#import "DYTool.h"
```
